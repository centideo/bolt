A restfull database connector!

This is Bolt, a high performance cross platform C++ restful database connector. It is built around the well known REST-based architecture and its use HTTP requests to post data (create and/or update) read data, and delete data from databases.

Latest Stable Version


There is no stable version available yet. You can obtain the source which is available in our github repository.

Contributing to the Effort


If you want to participate in the coding effort, the Bolt development project is hosted in github. Documentation contatins further instaurctions please read it before making changes to the library. For a great source of projects to tackle, see the Wishlist available in the documentaion.
You wouldn't want to go to the effort of creating a patch, only to find that someone had already checked in a change for that item.
